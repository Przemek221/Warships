Simple warship game in the windows console.

Gamemodes: Player vs Player, Player vs PC, PC vs PC

-Ships/Shots can be placed/fired automatically or manually
-Ships can be only horizontally or vertically oriented

-While placing the ship, the Player must know correct position of the ship(it's needed to think also about the size).
-Allowed size of the ship: 2,3,4,5.

Marks on the board:
* is the default/null state
o is the miss
X is the hit
[number] is the ship (e.g. the 3-tier ship is marked with '3' )
